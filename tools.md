# Tools Available to Atlas

- DuckDuckGo web search via `!search <query>` or automatic web lookup
- Local GGUF model via `llama-cpp-python`
- Memory storage for facts, preferences, and recent conversation at ~/.AtlasAI/memory.jsonl
- Chat history saving with `!savechat` and `!chatlog`
- Runtime model swapping with `!loadmodel <path>` or `!model <path>`
